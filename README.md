API tester
==========

Simplicit&eacute;&reg; REST APIs tester based on the [Unirest](https://www.npmjs.com/package/unirest) lib.

Install
-------

```bash
npm install
```

Lint
----

```bash
npm run lint
```

Run
---

```bash
npm run start
```
